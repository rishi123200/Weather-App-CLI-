Overview🔸

The Weather App (CLI) is a command-line Python application that allows users to check weather information for a specific city. The application retrieves and displays weather details such as temperature, weather conditions, humidity, and wind speed in a simple text-based interface.

Objectives📄

- To provide real-time weather information through a command-line interface.
- To demonstrate the use of APIs in Python.
- To improve skills in handling user input and data processing.

Features🔍

- Search weather by city name
- Display current temperature
- Show weather conditions
- Display humidity and wind speed
- Simple command-line interface

Technologies Used🛰️

- Python 3
- Weather API
- Requests Library

Applications📨

- Daily weather monitoring
- Travel planning
- Educational projects
- Learning API integration

Advantages📈

- Easy to use
- Fast weather updates
- Lightweight application
- Beginner-friendly project

Future Enhancements🏔️

- 7-day weather forecast
- Graphical User Interface (GUI)
- Multiple city comparison
- Weather alerts and notifications

Conclusion🔹

The Weather App (CLI) is a practical Python project that demonstrates how to integrate external APIs and display real-time data. It provides users with quick access to weather information through a simple command-line interface.
